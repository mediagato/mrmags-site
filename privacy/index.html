Privacy

What we know about you.

Plain-language version: almost nothing. The desktop app keeps your data on your machine. The website does what every website does — sees who visits — and that's it. No third-party trackers, no Google Analytics, no Meta pixel, no ad networks, no “cookies for personalization.”

The desktop app

Mr. Mags runs on your Mac or Windows machine. The brain — every memory you save, every pattern, every state value — is a database file in your user data folder.

  • Mac: ~/Library/Application Support/Mr. Mags/brain/
  • Windows: %APPDATA%\Mr. Mags\brain\

That folder never gets uploaded anywhere. There is no “cloud sync” today. There is no background telemetry that reports the contents of your memories. The MCP relay between Claude Desktop and Mr. Mags is a localhost-only HTTP API on 127.0.0.1:11436 — your computer talking to itself.

We’re building this so that the moment we ship cross-device sync as a paid feature, it stays opt-in and end-to-end encrypted. Your conversations are between you and the AI you chose; we are not in the middle.

This website

The site is hosted on Cloudflare Pages. Every page request hits Cloudflare's servers, which keep standard access logs (timestamp, IP address, user-agent, URL) for a short window — that's how the internet works, not specific to us.

On top of that, we run a download counter: when you click a Download button, our server logs the filename, a salted SHA-256 hash of your IP (so we can roughly count unique downloads without ever holding the IP itself), an 80-character snippet of your browser's user-agent, and Cloudflare's two-letter country code. Then it 302-redirects you to the actual binary on our CDN. Numbers come back at /api/stats — total, last-7-days, breakdown by file. Public, in case you're curious.

That's it. No Google Analytics. No Meta pixel. No Hotjar. No newsletter signup popup. No retargeting cookies. No ad networks.

The teacher verifier

If you submit your school email at /teachers, we store:

  • Your email address
  • A pending verification token (24-hour TTL, one-time use)
  • Once you click the link: a permanent license code we generated for your account

That's stored in a Cloudflare D1 database (SQLite-compatible, in our account). The verification email itself goes through Resend — one transactional email provider we use, who sees the send-event but doesn't keep the message contents long-term per their policy.

We don't sell, share, license, or otherwise hand off your email. We don't add it to a newsletter. If you want it deleted, email hello@mrmags.org and we delete the row. Real human, real keyboard.

What we'd like to add later (and how)

We’re considering an anonymous heartbeat from the desktop app — once a week it would ping the server with an install UUID, the version number, and the OS family (mac/win), so we know roughly how many people are using it. Zero PII, zero memory content, zero anything you typed. When that ships, it will be:

  • Disclosed plainly here, with the exact data shape
  • Toggled off in Settings with one click
  • Visible to you — clicking the toggle once shows the literal request body that would be sent

Until that ships, the desktop app is silent. It does not call home.
