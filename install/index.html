Install

Three minutes from download to done.

Mr. Mags is built by one person. I'm not enrolled in Apple's $99/year Developer Program or Microsoft's code-signing program yet — that comes when the app earns it. Until then, both Mac and Windows will warn you on first launch. That's expected, not a bug. This page tells you exactly what you'll see, why it happens, and the fastest way through it. After that the app sits quietly in your menu bar or system tray, gives Claude a memory, and leaves you alone until you need it.

A note from the developer: I know the security warning is annoying. I'm working on getting the app properly signed — it's on the roadmap. In the meantime, every path below is a one-time thing. Once you're past it, Mr. Mags opens normally forever. If anything goes sideways, email me: hello@mrmags.org. I read every message.

M On a Mac

  1. Download the installer.

    Download Mr-Mags.dmg →

    Universal binary. Works on Apple Silicon (M1/M2/M3/M4) and Intel.

  2. Open the DMG. Double-click the file in your Downloads folder. A small window opens with the Mr. Mags icon and a shortcut to your Applications folder.
  3. Drag Mr. Mags into Applications. Just like installing any other Mac app.
  4. The first launch — read this part.

    Mac will block Mr. Mags because the app isn't code-signed yet. This is a one-time hurdle. Pick the path that matches your macOS version — all three end with the app opening normally.

    Why this happens: Apple requires developers to pay $99/year and submit apps for notarization before macOS trusts them silently. Mr. Mags is an independent project — I'm not enrolled yet. The app itself hasn't changed; Mac just doesn't recognize my name. Any of the three paths below is a one-time permission grant. After your first successful launch, double-clicking Mr. Mags works forever.

    Path A — Right-click → Open macOS 14 Sonoma and earlier

    • In Applications, right-click Mr. Mags (or Control-click). Choose Open.
    • A dialog appears asking if you're sure. Click Open. Done.

    Path B — Privacy & Security macOS 15 Sequoia — may not work reliably

    • Double-click Mr. Mags. It gets blocked. Click Done to dismiss the dialog.
    • Open System Settings → Privacy & Security. Scroll to the Security section.
    • Find the line "'Mr. Mags' was blocked to protect your Mac." Click Open Anyway.
    • Enter your password. Click Open Anyway one more time. Done.
    • If the "Open Anyway" button never appears, use Path C — Sequoia 15.4+ sometimes skips showing it.

    Path C — Terminal one-liner works on every macOS version, most reliable

    • Open Terminal: press ⌘ Space, type Terminal, hit Enter.
    • Paste the line below and press Enter. It runs silently — no output means it worked.
    xattr -d com.apple.quarantine "/Applications/Mr. Mags.app"
    • Double-click Mr. Mags. It opens immediately, no warning.

    What that command does: macOS tags every internet download with a "quarantine" flag. That one line removes the tag. The app is bit-for-bit identical to what you downloaded — nothing about it changes. This is the same action as "Open Anyway," just done directly instead of through a GUI that Sequoia sometimes hides.

  5. Mr. Mags introduces himself.

    A small window pops up: "Hi, I'm Mr. Mags. I just gave Claude a memory." Click Got it. The Mr. Mags window opens with a cheat sheet and a memory browser.

  6. Quit and reopen Claude Desktop.

    If Claude Desktop was already running when you installed, quit it (⌘Q) and reopen it. Claude picks up the new memory tools on launch. If it wasn't running, just open it now.

  7. Try it.

    In Claude, type: "Remember that I teach 9th grade biology."

    Then type: "What grade do I teach?"

    Claude should answer correctly. That's it. Mr. Mags is working.

Where the memory file lives: ~/Library/Application Support/Mr. Mags/brain/ Nothing leaves that folder. No cloud, no account, no telemetry.

W On Windows

  1. Download the installer.

    Download Mr-Mags-Setup.exe →

  2. Run the installer. Double-click Mr-Mags-Setup.exe in your Downloads.
  3. The SmartScreen warning — this is the part to read.

    Windows will pop up a blue window: "Microsoft Defender SmartScreen prevented an unrecognized app from starting."

    Click the small "More info" link. The window expands. Click the "Run anyway" button that appears.

    Why this happens: we haven't bought a Windows code-signing certificate (~$300/yr) — early access. SmartScreen flags any installer it hasn't seen signed before. "Run anyway" is the right button. The installer is the one you just downloaded from mrmags.org — Windows just doesn't know us yet.

  4. Step through the installer.

    Pick whether to install for "just me" or "all users" (just me is fine). Accept the default install location. Let it create a desktop shortcut. Click Install. Wait a few seconds. Click Finish.

  5. Mr. Mags introduces himself.

    A small window pops up explaining what just happened. Click Got it. The Mr. Mags window opens with a cheat sheet and a memory browser.

  6. Quit and reopen Claude Desktop.

    If Claude Desktop was already running, fully exit it (right-click the Claude tray icon → Quit) and reopen it. Claude picks up the new memory tools on launch.

  7. Try it.

    In Claude, type: "Remember that I teach 9th grade biology."

    Then type: "What grade do I teach?"

    Claude should answer correctly. That's it. Mr. Mags is working.

Where the memory file lives: %APPDATA%\Mr. Mags\brain\ Nothing leaves that folder. No cloud, no account, no telemetry.
